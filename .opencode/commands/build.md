Load and follow the `incremental-implementation` and `test-driven-development` skills using the native `skill` tool.

Pick the next pending task from the plan. For each task:

1. Read the task's acceptance criteria
2. Load the relevant code, tests, and local patterns
3. Write a failing test for the expected behavior
4. Implement the minimum code to make it pass
5. Run the full test suite for regressions
6. Run the build to verify compilation
7. Summarize the completed slice and move to the next task

If any step fails, switch to the `debugging-and-error-recovery` skill.
